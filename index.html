Como funciona a comissão?

A cada depósito dos seus indicados, o DePix App cobra uma taxa de 2%. Você receberá 50% dessa taxa como comissão.

Exemplo: um indicado deposita R$ 1.000 → taxa DePix App: 20,00 DePix → sua comissão: 10,00 DePix.

O que é Volume Total?

O Volume Total é a soma de todos os depósitos PIX realizados pelos usuários que você indicou desde o início.

Comissões a receber

Comissões a receber é o total acumulado de todas as suas comissões, menos os valores já pagos.

Esse valor cresce conforme seus indicados fazem depósitos PIX.

Comissões recebidas

Este é o total de comissões que já foram pagas a você desde o início do programa.

Os pagamentos são feitos em DePix, diretamente para o endereço Liquid que você informar ao solicitar. Você pode acompanhar cada pagamento na tela Minhas Comissões.

Importante

Você só pode solicitar pagamento de comissões 1 vez por mês.

O que é a Callback URL?

A Callback URL é um endereço do seu sistema que recebe notificações automáticas quando um pagamento muda de status (ex: foi pago, expirou).

Quando configurada, enviamos uma requisição para essa URL com os dados do pagamento. Assim seu sistema pode atualizar pedidos automaticamente, sem precisar consultar a API.

Para quem é: desenvolvedores que querem integrar o DePix com loja virtual, app ou sistema próprio.

O que é a Redirect URL?

A Redirect URL é a página para onde o cliente é levado após concluir o pagamento.

Por exemplo, pode ser uma página de "obrigado pela compra" ou de acompanhamento do pedido. Se não configurada, o cliente permanece na página de pagamento do DePix.

Para quem é: quem quer oferecer uma experiência personalizada ao cliente após o pagamento.

O que é Metadata?

Metadata são dados extras que você pode anexar a um produto em formato JSON. Esses dados são retornados junto com o checkout nas consultas da API e nos webhooks.

Exemplo: guardar o SKU do produto, ID interno do seu sistema, ou qualquer informação que ajude a identificar a venda do seu lado.

Para quem é: desenvolvedores que integram o DePix com sistemas próprios e precisam rastrear dados adicionais por checkout.

Dicas para sua imagem

  • Use imagens quadradas (1:1) para melhor resultado
  • Tamanho recomendado: pelo menos 360×360 pixels
  • Formatos aceitos: JPG, PNG, WebP
  • A imagem será redimensionada e otimizada automaticamente
  • Prefira fundo branco ou transparente para logos
  • Evite texto pequeno — pode ficar ilegível no tamanho final

O que é a Expiração?

A Expiração é o tempo que o cliente tem para pagar depois de abrir o checkout. Quando esse prazo acaba, o QR code PIX expira e a cobrança é marcada como expirada automaticamente.

Se não escolher, usamos o padrão de 20 minutos, que funciona bem para a maioria dos casos.

Prazos curtos (5–10 min) evitam cobranças abandonadas. Prazos longos dão mais margem para clientes que demoram a abrir o app do banco.

O que é a Expiração da chave?

Define por quantos dias a chave de API continuará válida após ser criada. Passado esse prazo, a chave deixa de funcionar automaticamente e sua integração precisará de uma nova.

Se deixar em branco, a chave não expira. É recomendado definir um prazo por segurança — assim chaves esquecidas não ficam ativas para sempre.

Máximo: 365 dias. Você pode revogar manualmente a qualquer momento na lista de chaves.

O que é o Slug?

O slug é a parte final da URL de checkout do produto (ex: pay.depixapp.com/seuusuario/camiseta-preta).

Ele é gerado automaticamente a partir do Nome do produto. Você só precisa editá-lo manualmente se quiser personalizar a URL.

Use apenas letras minúsculas, números e hifens (2–60 caracteres).

Instalar o DePix App

Adicione o DePix na tela inicial do seu celular para acesso rápido, como um app nativo.

No iPhone / iPad (Safari)

  1. Toque no botão Compartilhar () na barra inferior do Safari
  2. Role as opções e toque em Adicionar à Tela de Início
  3. Confirme tocando em Adicionar

O DePix vai aparecer na sua tela inicial como um app comum.

No Android (Chrome)

  1. Toque no menu ⋮ (3 pontinhos) no canto superior
  2. Escolha Adicionar à tela inicial ou Instalar app
  3. Confirme tocando em Instalar
