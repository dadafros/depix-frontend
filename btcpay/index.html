Recebeu? É seu.

Pix tradicional pode ser estornado pelo banco do pagador por até 80 dias — é o MED. Com DePix, quando o status chega em "Concluído", os tokens estão na sua carteira e a transação é irreversível. Sem estorno. Sem surpresa.

Dúvidas

Perguntas frequentes

Meu cliente precisa saber que é criptomoeda?

Não. Para o seu cliente, é um Pix normal — ele abre o app do banco, escaneia o QR code e paga. Não aparece nada de criptomoeda na tela dele. A conversão para DePix acontece automaticamente na plataforma.

O que é BTCPay Server?

BTCPay Server é um sistema de pagamentos open-source para quem vende online. Ele permite receber criptomoedas sem depender de nenhuma empresa. Nosso plugin conecta o BTCPay à API do DePix App, adicionando Pix como método de pagamento no checkout da sua loja.

Como o status do pedido é atualizado no BTCPay?

Assim que o pagamento é confirmado, a API do DePix App envia um webhook para o seu servidor BTCPay com o evento checkout.completed. O plugin processa o evento e marca o pedido como pago automaticamente. Você não precisa verificar manualmente.

Posso testar antes de usar em produção?

Sim. Ao criar uma API key do tipo "test" (sk_test_...), todos os checkouts criados são de teste. Você pode simular um pagamento pela API sem movimentar dinheiro real. Quando estiver satisfeito, troque para uma chave live. Veja a documentação do sandbox para detalhes.
